# Entity Logo Rollout

## Objetivo

Integrar representación visual consistente de entidades en la aplicación.

## Componentes base

EntityLogo  
EntityBadge

## Fuente de logos

/public/logos/entities/

## Orden de integración

1 Versus  
2 Progresivos  
3 Rankings  
4 Results  
5 Depth  

## Regla principal

Nunca cargar logos desde APIs externas en runtime.
Siempre usar assets locales.
