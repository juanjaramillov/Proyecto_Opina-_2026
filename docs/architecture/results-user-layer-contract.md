# Results User Layer Contract

## Objetivo

Definir qué tipo de lectura debe exponerse al usuario final en la capa Resultados.

## Qué sí debe mostrar Resultados

- rankings visibles
- líderes por subcategoría
- tendencia simple
- evolución básica
- comparación personal con otros
- highlights resumidos por módulo

## Qué no debe mostrar Resultados

- segmentación avanzada completa
- cruces profundos multi-variable
- exploración libre tipo BI
- exportación detallada
- toda la profundidad de atributos disponible para Inteligencia

## Regla

Resultados debe explicar qué está pasando sin regalar toda la inteligencia del sistema.
