# OPINA+ — SOURCE PRIORITY OBLIGATORIA

Este documento define la jerarquía oficial de fuentes para construir el deck comercial de Opina+ en NotebookLM.

## REGLA CENTRAL
NotebookLM no debe conciliar, fusionar ni resumir la estructura oficial del deck para parecerse a PDFs anteriores o referencias visuales.

## JERARQUÍA OBLIGATORIA DE FUENTES
Cuando exista conflicto entre fuentes, el orden de prioridad obligatorio es el siguiente:

1. **Opina+_Brand_Guardrails_Obligatorio**
2. **Opina+_Deck_Master_20_Slides_Oficial**
3. **Opina+_Narrativa_Comercial_Oficial**
4. **Opina+_Industrias_Casos_de_Uso_Oficial**
5. **PDFs visuales de referencia**

## REGLAS ESTRICTAS
- La estructura oficial del deck es de **20 slides exactas**.
- No se permite reducir, fusionar, resumir ni agrupar slides.
- Los PDFs visuales **NO** definen cantidad de slides.
- Los PDFs visuales **NO** definen paleta de color.
- Los PDFs visuales sirven solo para:
  - composición
  - layout
  - ritmo visual
  - jerarquía gráfica
  - estilo editorial B2B premium
- La paleta corporativa debe obedecer únicamente el documento de guardrails de marca.
- Si un PDF antiguo contradice el deck maestro, manda el deck maestro.
- Si un PDF usa colores aproximados, manda el documento de marca.
- Si una referencia visual sugiere una slide menos, se mantiene igualmente la estructura oficial de 20 slides.

## OBJETIVO DE CADA FUENTE
### 1. Brand Guardrails
Define color, iconografía, estilo visual y reglas negativas.

### 2. Deck Master 20 Slides
Define estructura exacta, orden, propósito y contenido por slide.

### 3. Narrativa Comercial
Refuerza mensajes de venta, tono, problemas, diferenciación y lógica comercial.

### 4. Industrias y Casos de Uso
Amplía industrias, ejemplos y razones de valor por sector.

### 5. PDFs Visuales
Aportan inspiración compositiva, no estructura ni reglas.

## RESULTADO ESPERADO
NotebookLM debe producir un deck que:
- respete exactamente 20 slides
- use exactamente la paleta Opina+
- mantenga un look premium, limpio, tecnológico y comercial
- no mezcle versiones antiguas
- no comprima el contenido para parecerse a decks previos

## INSTRUCCIÓN FINAL OBLIGATORIA
Desarrollar el deck respetando 1:1 la estructura del documento maestro de 20 slides. No fusionar ni reinterpretar cantidad de slides. Obedecer estrictamente la jerarquía de fuentes definida en este documento.
