# Correcciones sobre Opina_Comercial_v6

Tomar como base la versión v6, pero ejecutar una nueva pasada completa de corrección visual, editorial y de branding.
Esta es una segunda ronda de ajuste. No rehacer la estructura desde cero. Mantener las 20 slides exactas, pero corregir identidad de marca, uso del gradiente, calidad de ilustraciones y errores editoriales.

## CAMBIOS GLOBALES (OBLIGATORIO)
- La versión final debe seguir teniendo exactamente 20 slides. No fusionar, resumir, eliminar ni agregar slides.
- Respetar la jerarquía oficial de fuentes: 1) Brand Guardrails, 2) Deck Master 20 Slides, 3) Narrativa Comercial Oficial, 4) Industrias y Casos de Uso Oficial, 5) PDFs previos como referencia visual.
- Usar exclusivamente el logo oficial correcto de Opina+ que está adjunto en el proyecto en TODA la presentación. No usar variantes antiguas, isotipos equivocados, logos simplificados ni reinterpretados.
- La portada ya está más cerca de lo correcto. Replicar ese mismo estándar de logo oficial en todas las demás slides donde aparezca marca, especialmente en cierres, headers y slide 20.
- Hacer una pasada completa para detectar y reemplazar cualquier logo incorrecto o variante visual inconsistente del logo.
- El uso de color corporativo para destacar palabras o frases NO debe hacerse dejando una palabra azul y otra verde como bloques separados. El destaque debe hacerse con el gradiente oficial completo azul → teal aplicado como un solo tratamiento visual continuo.
- Aplicar el gradiente oficial de manera más elegante, continua y premium en palabras o frases destacadas. Nada de alternar palabra por palabra con colores planos distintos.
- Usar el gradiente oficial azul corporativo (#2563EB) → teal corporativo (#10B981) con moderación y solo donde realmente agregue jerarquía: portada, slide 20, headers importantes, conceptos hero, números grandes o nodos centrales.
- No abusar del gradiente. Mantener la presentación respirable, editorial y seria.
- Revisar toda la presentación para eliminar cualquier azul o verde fuera de la paleta oficial. No usar colores aproximados.
- Mejorar de forma general la calidad de los diseños, diagramas e ilustraciones. Hoy varias piezas están funcionales, pero no se sienten 100% premium ni suficientemente refinadas.
- Tomar como referencia las ilustraciones más logradas de las presentaciones previas y reutilizar esa dirección visual donde sea superior a v6, siempre adaptándola al sistema oficial de marca y al deck maestro.
- No copiar de forma literal una visual anterior si rompe coherencia. La instrucción es rescatar lo mejor de versiones previas y elevarlo dentro de un sistema visual único y consistente.
- La dirección visual de ilustraciones debe ser: premium, tecnológica, limpia, editorial B2B, moderna, minimalista y corporativa. Nada genérico. Nada que parezca template común. Nada caricaturesco.
- Refinar especialmente volumen, profundidad, bordes, conectores, proporciones, glow, opacidades, pesos de línea y composición de las ilustraciones para que se vean más maduras y mejor terminadas.
- Hacer una limpieza editorial completa de toda la presentación. Corregir cualquier typo, texto extraño, mezcla rara de idiomas, placeholder, término inventado o label corrupto.
- No dejar errores de copy en ninguna slide. Todo debe verse listo para presentar comercialmente.
- Unificar mejor el sistema de títulos, bordes, tarjetas, iconografía, diagramas y sombras suaves para que todo se sienta como un único deck madre y no como mezcla de soluciones distintas.
