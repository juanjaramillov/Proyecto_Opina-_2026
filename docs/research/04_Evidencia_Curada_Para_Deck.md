# OPINA+ — EVIDENCIA CURADA PARA DECK

## PROPÓSITO
Este documento consolida la evidencia que sí conviene usar para la futura presentación comercial de Opina+.

Está construido con un criterio estricto:
- usar solo hallazgos suficientemente trazables y comercialmente útiles;
- separar evidencia principal de respaldo;
- dejar fuera claims débiles, ambiguos o sobreinterpretados;
- y preparar una base limpia antes de construir el deck.

## 3. CLAIMS DESCARTADOS

Estos hallazgos no conviene usarlos en la futura PPT mientras no exista sustento más limpio o directo.

| Claim descartado | Por qué se descarta |
|---|---|
| Tasas de respuesta cayeron de 30% a 18% y 70% abandona por fatiga | Dependía de fuentes débiles o no importadas en la versión final. |
| 40% de registros contaminados y 4–5% de fraude sintético | No quedó suficientemente amarrado a una fuente importada fuerte y directa. |
| NPS cayó al 8º lugar en prioridad de CX | La idea general sirve, pero esa cifra exacta no quedó bien trazada. |
| 29% de churn silencioso | Quedó sin respaldo suficientemente limpio en el set final. |
| 69,5% de error entre memoria declarada y comportamiento real | Interesante, pero no quedó sostenido por fuente importada fuerte y directa. |
| Marcas propias crecen 3,7% vs 1,1% en Chile | La cifra exacta no quedó confirmada en el set final importado. |
| Cifras ilustrativas de dashboards del material interno | Son ejemplos inventados para explicar el producto, no evidencia real de mercado. |
| Narrativa de “Chile en colapso” por quiebras | Exagera el contexto y contradice el marco más equilibrado de Hacienda. |
| Meta de turismo 2030 como evidencia actual | Es una proyección gremial, no un comportamiento probado del consumidor hoy. |
| Blogs menores, YouTube, comparadores de software o páginas promocionales | Restan credibilidad frente a una audiencia ejecutiva. |

## 4. NOTAS DE USO PARA EL DECK

### Qué sí conviene hacer
- Usar los hallazgos principales para abrir problema, urgencia y oportunidad.
- Combinar evidencia global de contexto con evidencia chilena para aterrizar la oportunidad local.
- Usar la evidencia secundaria como respaldo, no como titular.
- Mantener el framing de “mercado ajustado y racionalizado”, no “mercado colapsado”.

### Qué no conviene hacer
- Basar el deck en cifras “perfectas para vender” si no están totalmente trazadas.
- Sobrecargar la historia con demasiados números.
- Convertir el argumento en una crítica abstracta a encuestas, NPS o dashboards sin aterrizar el vacío concreto que deja el mercado.
- Presentar ejemplos internos ilustrativos como si fueran hallazgos reales.

### Precaución importante
El dato de **36% a 6%** es útil como benchmark global fuerte, pero conviene tratarlo como **contexto de apertura** y no como único gran ancla del deck, porque entra vía `Opina_Maestro_Comercial_v8.pdf` y no como una fuente primaria independiente importada en esta etapa.

## 5. LOS 8 HALLAZGOS QUE MÁS CONVIENE LLEVAR AL DECK

1. La encuesta intrusiva perdió efectividad estructural.  
2. La lealtad pasiva a las marcas se rompió.  
3. Las marcas propias ya compiten como primera opción.  
4. El mercado chileno ya no puede leerse con promedios país.  
5. La simplicidad y la baja fricción son parte de la respuesta.  
6. Chile ya tiene infraestructura para captura masiva y mobile-first.  
7. El cliente B2B está más preparado para inteligencia avanzada.  
8. El mercado está ajustado: leer tarde una preferencia cuesta más.
